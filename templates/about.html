Welcome to Prompt Analysis Tool

The Prompt Analysis Tool is a web application that uses OpenAI's GPT-3.5 Turbo model to analyze chat prompts and provide scores based on specific criteria.

Objective

The main goal of this project is to offer a user-friendly web tool for analyzing chat prompts using OpenAI's GPT-3.5 Turbo model. It helps ensure clear and effective communication with the language model.

Features

  • Input OpenAI API Key: Securely provide your API key to access the language model's capabilities.
  • Analyze Chat Prompt: Input a chat prompt to analyze system and user instructions, cues, and context information.
  • Prompt Scoring: Assigns a score (1-100) based on clarity, specificity, and context.
  • Generate New Prompt: Creates a new prompt that meets required parameters.

How to Get an OpenAI API Key

  1. Create an OpenAI Account: If you haven't already, go to the OpenAI website and create an account.
  2. Log in to your Account: Use the credentials you created during the registration process to log in to your OpenAI account.
  3. Navigate to Personal: Once logged in, find the section view API Keys .
  4. Generate an API Key: There should be an option to generate an API key. This key will serve as your authentication token to access OpenAI's services.
  5. Follow the Key Generation Process: The exact steps might vary, but generally, you'll need to click on a "Generate API Key" or similar button. The system might prompt you to provide a name for the key or associate it with a specific project.
  6. Retrieve the API Key: Once generated, your API key will likely be displayed on the screen. It might be a long string of characters. This key is sensitive and should be kept secure. Copy that Key.

How to Use

  1. Access the app at http://127.0.0.1:8080/.
  2. Enter OpenAI API key in password-protected input.
  3. Input chat prompt and see the analysis.
